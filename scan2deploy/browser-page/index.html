Enterprise Browser Page

This page allows the user to configure the Datalogic Enterprise Browser.

Import settings from file​

The Import settings section allows you to provide a path to a file on the device that contains Datalogic Enterprise Browser configuration. You can read more on this topic on the Export/Import Settings page within the documentation for the Datalogic Enterprise Browser.

Allowed sites​

The Allowed sites section controls which websites are allowed access from the browser. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Allowed Sites page.

Favorites appearance​

The Favorites appearance section controls the contents of the Favorites screen. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Favorites Settings page.

Browser Display​

The Browser Display section controls the contents of the display when browsing. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Display Settings page.

Browser Security​

The Browser Security section controls web page access to sensitive information on the device. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Security Settings page.

Browser Navigation​

The Browser Navigation section controls web page navigation when browsing. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Navigation Settings page.

Browser Administration​

The Browser Administration section controls access to the Browser configuration settings. This section exposes settings found within the Datalogic Enterprise Browser app, which are documented on the Browser Administration page.
