UI Controls

This section explains the Visual Formatter UI with its usages and shortcuts.

The nodes on the left (in the Parameters and Operations sections) can be dragged and dropped in the flowchart on the right in order to add a specific component into the chart. You can connect an output of a node with an input of a different node with the left mouse by creating connections. with All the different components are explained in the Basic Concepts section.

At the top there are several buttons that allow to execute some actions:

  • **New Chart**: Used to reset the current chart to its default (showing only barcodeStringIn and barcodeStringOut without connections);
  • **Edit Chart Info**: Opens a dialog to edit some info about the chart, such as its name (Item Name field) or the name of the project;
  • **Undo/Redo**: Used to undo/redo the operations executed in the chart (adding of new node or connection, removal of nodes, etc...). Undo shortcut is *Ctrl+Z*, redo shortcut is *Ctrl+Y*;
  • **Delete Selected**: Removes the selected nodes and connections. In order to select a node you can just click on it. You can select multiple nodes and connections by keeping *Ctrl* pressed while clicking on the items or by keeping *Shift* pressed together with the left mouse button and move the cursor in order to create a rectangle region including the items that you want to select;
  • **Run Debugging**: Used to test the currently created chart. It opens a dialog asking to set the value of all the inputs available in the chart. After doing it and confirming the action, all the visual script is executed and the results are available inside the output nodes. It is also possible to see the intermediate results by hovering over the connections between the nodes or by clicking on the **Show All Intermediate Values** button. To hide all debugging values you can click on the **Clear Debugging Info** button.

Some shortcuts ara available inside the Visual Formatter UI:

  • Ctrl+Z: Undo;
  • Ctrl+Y: Redo;
  • Ctrl+A: Select all nodes and connections currently included in the chart;
  • ESC: De-select all;
  • Ctrl+D: Duplicate the selected nodes and connections. The connections will be duplicated only if they are between nodes that are both selected.
