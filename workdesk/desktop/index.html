Desktop Settings

Overview​

The Desktop Settings group is a page which controls the content and appearance of the desktop wallpaper and shortcuts as well administrator access.

Wallpaper Settings​

The following settings are available for configuring the display of the desktop wallpaper:

SettingDescription
Use custom wallpaperWhen disabled, the desktop displays the same wallpaper as the device display. When enabled, the desktop uses the following controls to determine the wallpaper appearance.
Custom imageThis selects a custom wallpaper image. See the following section for details.
Image fitThis determines how the wallpaper image is placed on the desktop. See the following section for details.
Background colorThis selects any background that may be seen with the wallpaper. See the following sectionfor details.

Selecting a Wallpaper Image​

To select an image for the background wallpaper, tap the Custom image item. This presents a window which lets you select an image to use as the wallpaper. The images shown to select from are taken from the public "Downloads" and "Pictures" folders on the device. These are the locations where image files are typically downloaded from a browser (or similar application), and can be viewed by launching the standard Files application on the device. Supported image file types are PNG, JPG and BMP. To remove a wallpaper image, so that the entire background is a single color, tap the No image button.

Once a wallpaper image is selected, the background color and size are automatically adjusted, as needed, to match the contents of the image. The background color is changed to a value which best blends with the edges of the image. This works best with an image that contains a solid color at its edges. The image size may also be adjusted to use stretch sizing to prevent clipping an image which is larger than the screen. The controls for size and background color can still be altered indepedently after an image is selected.

Selecting a Background Color​

To select the background color, tap the Color button. This displays a dialog with Red/Green/Blue slider controls which can be adjusted to change the hue of the background color. The background color is visible in any area of the wallpaper image which is transparent or outside the image. The dialog also includes a toggle to remove any background image that may currently exist.

note

Whenever the wallpaper image or background color is changed, the shortcut label color is automatically changed to white or black. The exact color is chosen based on an attempt to keep a high contrast between the background and text colors. The label color may still be adjusted to a different color within the Shortcut control group.

Fitting a Wallpaper Image​

The following options are available for controlling how a wallpaper image will be fit within the desktop display area:

OptionDescription
CenterThis displays the wallpaper image as centered, without any resizing.
Stretch To FitThis displays the wallpaper image as centered, and resized to reach the nearest display edge. The original aspect ratio of the image is preserved, which can result in bands of the background color visible on the horizontal or vertical edges of the image.
Stretch To CropThis displays the wallpaper image as centered, and resized to reach the farthest display edge. The original aspect ratio of the image is preserved, which can result in cropping of the horizontal or vertical edges of the image.
FillThis displays the wallpaper image as centered, and resized to fill the entire display. The original aspect ratio of the image is not preserved.

Shortcut Appearance​

The following settings are available for configuring the display of the desktop shortcuts:

SettingDescription
Show standard shortcutsWhen enabled, standard shortcuts are added to the start of the shortcut list. The standard shortcuts are: Settings, Chrome, Files.
Shortcut sizeThis slider control determines the size (small, medium or large) of the icon and label used to display shortcuts.
Shortcut label colorThis controls the color used to display the shortcut labels. Adjust the the Red/Green/Blue slider controls to change the hue of the label text color.

Administrator Password​

The administrator password controls access to the WorkDesk Settings. This includes the Settings window as well as the ability to alter the desktop shortcuts. It can be set from the Desktop Settings window by tapping its settings item.

When not set, access to changing WorkDesk settings is available at any time. When set, access is only granted once the password has been entered. Access continues until you leave Adminstration mode by clicking on the Administrator Access icon () in the far right of the taskbar.
