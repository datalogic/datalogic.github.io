To configure the decoder properties, you need to instantiate {% include man-link.html uri='com.datalogic.decode.BarcodeManager' %} and get a {% include man-link.html uri='com.datalogic.decode.configuration.ScannerProperties' %} instance.

{% highlight java %} BarcodeManager manager = null; ScannerProperties configuration = null; try { manager = new BarcodeManager(); } catch (DecodeException e) { e.printStackTrace(); } configuration = ScannerProperties.edit(manager); {% endhighlight %}

Properties are organized gerarchically Each property has the method set that requires a different argument depending on the property type (i.e.: int, boolean, String, etc...):

{% highlight java %} configuration.code39.enable.set(true); configuration.code39.enableChecksum.set(false); configuration.code39.fullAscii.set(true); configuration.code39.Length1.set(20); configuration.code39.userID.set('x'); if (configuration.qrCode.isSupported()) { configuration.qrCode.enable.set(false); } {% endhighlight %}

For the configuration to be actually applied, you must call the metdod {% include man-link.html uri='com.datalogic.decode.configuration.ScannerProperties#store(com.datalogic.device.configuration.PropertyEditor, boolean)' %}:

{% highlight java %} try { configuration.store(manager, true); } catch (ConfigException e) { e.printStackTrace(); return; } {% endhighlight %}
