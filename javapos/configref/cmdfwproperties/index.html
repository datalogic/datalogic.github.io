CMDFW Configuration Reference

Overview​

Datalogic CMDFW is configured with the cmdfw.properties file. This document covers each applicable configuration setting, the default configuration distributed with JavaPOS as well as any applicable API access. Each property in the cmdfw.properties file is a default and can be overridden with the appropriate command line switch.

Reference​

com.dls.jpos.cmdfw.version​

This property is obsolete and is no longer used. The version is now dynamically allocated during the build process and this property is not referred to. Changing this property has no effect.
