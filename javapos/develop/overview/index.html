Developing with JavaPOS

This section contains topics relating to developing wth the Datalogic JavaPOS API.

Java Platform#

Datalogic JavaPOS API is compiled on Java 8 using the -source=1.7 flag. This means that Datalogic JavaPOS is best suited in a Java 8 environment. Datalogic JavaPOS has been tested against the following Java platforms:

Oracle#

  • Java 7 SE JRE / JDK
  • Java 8 SE JRE / JDK
  • Java 11 SE JRE / JDK - Some adjustments to nrjavaserial may be required.

OpenJDK#

  • OpenJDK 8
  • OpenJDK 11
