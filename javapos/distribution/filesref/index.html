Microsoft Windows​

The following files are installed using the Datalogic JavaPOS installer on the Microsoft Windows platform.

brand.properties./Branding configuration file
dls.properties./Datalogic JavaPOS API-wide configuration file

IHSParser.csv./Information, Health and Statistics parsing configuration file
JavaPOS.jar./Datalogic JavaPOS API
jpos.xml./JPOS device registry configuration file

javax.usb.properties./SupportJarsjavax.usb Configuration (Linux only)

Win64Hid.dll./SupportJarsDatalogic Windows USB bridge (64-bit)
