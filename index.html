Android SDKs


Datalogic Android SDK

For Kotlin and Java app development.

Xamarin SDK

Xamarin (C#) interface to the SDK for Android devices. For development in Visual Studio.

Cordova SDK

Cordova plugin for use in Ionic and Phonegap apps.

React Native SDK

React Native module for use in React Native apps.

Handheld Core SDK

Interface an Android app with a Datalogic Bluetooth® hand held scanner.

Android Configuration


Scan2Deploy Studio

Software for configuring Android devices using barcodes.

DXU

Windows desktop software for configuring Android and WEC7.

OEMConfig

Software for configuring Datalogic specific settings on Android devices. Used by 3rd party EMMs.

Wi-Fi QR Code Generator

Create standard and encrypted Wi-Fi QR codes.

Android Enterprise QR Code Generator

Create standard and encrypted Android Enterprise Enrollment QR codes.

Memor 10 Ethernet Adapter Service Utility

Android application that manages the Memor 10 connection to the Ethernet cradle.

Datalogic WiFi Guard

Android application that assists in improving Wi-Fi network performance and diagnosing connection problems.

Windows CE


Windows CE SDK

Datalogic SDK for Windows CE and Windows Mobile devices.

Datalogic CE Sync

A simple command line tool that makes use of the datalogic-ce-sync library.

Scan2Deploy CE

Software for configuring WEC7 devices using barcodes.

DXU

Windows desktop software for configuring Android and WEC7.

USBLAN

Driver for TCP/IP communication over USB connections.
