Good Read

Use it to enable good read notifications (LED, Green Spot, Vibrator).

The Good Read Overlay feature is an additional good read indication which displays an image on the screen when a scan is successful. The image is drawn over the existing screen contents for the same duration as other good read indicators.

Tap Good read overlay enable to enable the Good Read Overlay notification when a scan is successful.

Tap Good read overlay config to configure the color, the shape policy and the position of the image and to customize the overlays.

Overlay colorâ€‹

Specifies the color to use with the overlay. It is applied only when Preserve colors is disabled. The format is AARRGGBB (hexadecimal values of Alpha, Red, Green, Blue). The default value is FF00FF00.

Transparencyâ€‹

Drag the slider to set the overlay transparency.

Preserve colorsâ€‹

If enabled, the overlay image will retain its original colors and the overlay color will not be taken into account.

Black as transparentâ€‹

By default, the white areas in the overlay are treated as transparent. Enable this setting to treat the black areas as transparent.

Shape policyâ€‹

Specifies the shape to display for the overlay image. If Full screen is selected, the entire screen is filled with the selected color. The default value is Full screen. Select Custom to upload an overlay image.

Manage custom overlaysâ€‹

Allow to upload the overlay image to use when the policy shape setting is Custom. The file must have PNG format.

Portrait configurationâ€‹

Configures the position and size of the overlay in portrait orientation.

Set the Portrait position policy to Custom to configure the X/Y coordinate settings.

Set the Portrait display policy to Custom size to configure the width/height settings.

Landscape configurationâ€‹

Configures the position and size of the overlay in landscape orientation.

Set the Landscape position policy to Custom to configure the X/Y coordinate settings.

Set the Landscape display policy to Custom size to configure the width/height settings.
