Scanner Options

Use it to configure the good read tone and display notification:

Tap Scanner Options to customize the device scanning behavior.

Enable Scannerâ€‹

Enables the internal scanner.

Scanner/Camera Interoperabilityâ€‹

By default, the scanner is prevented from starting barcode scanning when the camera is in preview. This setting enables interoperability between the scanner and other cameras, allowing the user to scan barcodes even while another camera is in use.

Enhance DoFâ€‹

If enabled, allows to enhance the maximum reading distance.

Enable Picklistâ€‹

If enabled, it allows you to pick and decode a barcode from multiple barcodes printed close together, when the scan illumination intersects more than one barcode. Only the targeted barcode will be returned.

Enable Illuminationâ€‹

If enabled, it causes the scanner to turn on the illumination to aid decoding.

Enable Aimâ€‹

Enables the laser aim.

Display Mode (Antireflex)â€‹

Optimizes the reading of a barcode from a screen.

Scan Modeâ€‹

Selects the scan operating mode for the reader.

Singleâ€‹

When the trigger is pulled, scanning is activated until one of the following occurs:

  • a label has been read
  • the trigger is released
  • the decode timeout has elapsed.

Hold Multipleâ€‹

When the trigger is pulled, the device scans barcodes until the trigger is released or the decode timeout has elapsed.

Pulse Multipleâ€‹

When the trigger is pulled, continuous scanning is activated until the decode timeout has elapsed or the trigger has been released and pulled again.

Always Onâ€‹

No trigger pull is required to read a barcode. Scanning is continually on.

Enable Target Modeâ€‹

If enabled, when the scan button is pressed, the scanner will project an aiming pattern to assist in centering over the barcode before scanning. Tap Target mode to select the desired targeting behavior.

Spot Timeoutâ€‹

Scanning takes place after a programmable time upon pressing the scan button. Tap Target timeout to set the duration of the spot.

Release Scanâ€‹

Scanning takes place after the scan button is released. Drag the Release scan timeout slider to set the scanning timeout after releasing the scan button.

Decode Timeoutâ€‹

Drag the slider to set the maximum amount of time the scanner attempt to decode after target timeout (in case Spot Timeout is enabled) or after the scan button is pressed (in case Target mode is disabled).

Image Decode Timeoutâ€‹

You can share any image (picture, jpg/png, etc.) with the Datalogic Service to try to decode it. Drag the Image decode timeout slider to set the decode timeout applied with image decoding.

Enable Multi Scanâ€‹

If selected, the scanner captures multiple labels sequentially.

Number of Labelsâ€‹

Drag the slider to indicate the amount of labels required for a code to be confirmed.

Enable Single Notificationâ€‹

If selected, it enables indicators for each label, in order to get an intermediate notification for each label decoded.

Partial Resultsâ€‹

When multi scan mode is enabled, it selects the behavior when partial results are present.

Neverâ€‹

The decoding session is ended successfully and all the collected labels are returned together, only if the Number of Labels is reached.

Only on Timeoutâ€‹

The decoding session is ended successfully and all the collected labels are returned together, only if the decode timeout has elapsed.

Only on Releaseâ€‹

The decoding session is ended successfully and all the collected labels are returned together, only if the trigger is released.

Bothâ€‹

The decoding session is ended successfully and all the collected labels are returned together if the trigger is released or the decode timeout has elapsed.

Presentation Modeâ€‹

If enabled, the device starts to scan when motion is caught.

Aim Controlâ€‹

Enables the aimer while detecting motion.

Sensitivityâ€‹

Drag the slider to configure the motion detector's sensitivity.

External Scanner Sourceâ€‹

If enabled, allow barcode injection from an external scanner.

Standard Formatter Enableâ€‹

Executes the Standard Formatter on the injected barcodes.
