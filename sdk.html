--- layout: page title: Datalogic Android SDK Add-on ---

To install Datalogic Android SDK Add-on, please insert the following URL in Android Studio or Eclipse ADT or follow the detailed steps

{% highlight none %} https://datalogicadcsrl.github.io/android-sdk/addon.xml {% endhighlight %}

Android Studio

{% include info-box.html msg='Please notice that the following instructions are valid for Android Studio versions released on the stable, there is no support for the beta channel' %}

From Android Studio launch window, click on Configure > Android SDK Manager to open Android SDK Manager

{% include img.html src='install-sdk-as-configure.png' %} {% include img.html src='install-sdk-as-menu.png' %}

Select the tab SDK Update Sites and click the (plus) icon on the right-side toolbar

{% include img.html src='install-sdk-as-sdk-manager.png' %}

In the new window insert the following URL, optionally a name and the press OK

{% highlight none %} https://datalogicadcsrl.github.io/android-sdk/addon.xml {% endhighlight %} {% include img.html src='install-sdk-as-site-url.png' %}

Now select the tab SDK Platforms and check the checkbox on the bottom right Show Package Details. Under the section Android 4.1.2 (API 16) Datalogic SDK Add-on should appear. To install it, check Datalogic SDK Add-on checkbox and a small icon on the left should appear

{% include info-box.html level='warn' msg='Please notice that you must also install the Android API platform matching your Android version (i.e. API 16 is required to compile apps for Android 4.1.1)' %} {% include img.html src='install-sdk-as-before-install.png' %}

Click on the button Apply to install the selected packages. Once the installation is complete, the Status of the Datalogic SDK Add-on changes from Not installed to Installed

{% include img.html src='install-sdk-as-installed.png' %}

Create apps with Datalogic SDK

When creating a new project choose Datalogic Add-on SDK 1.0 as Minimum SDK option

{% include img.html src='install-sdk-as-create-project.png' %}

Compile with Datalogic SDK

Use Datalogic SDK in gradle scripts

Troubleshooting

Eclipse ADT

From Eclipse ADT menu, click Windows > Android SDK Manager to open Android SDK Manager

{% include img.html src='install-sdk-menu.png' %}

Click Tools > Manage Add-on Sites...

{% include img.html src='install-sdk-sdk-manager.png' %}

Select the tab User Defined Sites and click New...

{% include img.html src='install-sdk-site-url.png' %}

In the new window Add Add-on Site URL insert the following URL

{% highlight none %} https://datalogicadcsrl.github.io/android-sdk/addon.xml {% endhighlight %}
