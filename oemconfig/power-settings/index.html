Power & Charging

Settings#

General#

  • Suspend on external power: Sets the screen timeout for when the device is connected to an external power source (i.e. charging). The default value is 5 minutes.

  • Suspend on internal power: Sets the screen timeout for when the device is running on battery power. The default value is 1 minute.

Battery Saver#

  • Battery saver enable: Immediately enables battery saver mode on the device. The default value is false.
important

On Android 8, this setting may not persist through a reboot or connection to a charging source.

  • Battery saver auto enable: Sets the battery saver mode to be enabled when the battery falls below the specified trigger level. The default value is false.

  • Battery saver trigger level: Sets the trigger level (in battery percentage) for auto enabling the battery saver mode. The default value is 5.

note

May be scaled to the closest valid option depending on the Android version.

  • Battery saver auto disable: Sets battery saver mode to be automatically disabled when the battery is above 90%. The default value is true.
important

Battery saver auto disable is only available on Android 10+

Wakeup Sources#

  • Wakeup source: The wakeup source to enable/disable. The default value is None.

  • Enable: Enables the selected wakeup source. The default value is true.

Wireless Charging#

  • Charging profile: Represents the charging profile used to manage the charging preferences. This setting is persistent through reboots and factory resets. The default value is Speed and capacity preference.
