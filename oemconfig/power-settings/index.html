Power & Charging

Settingsâ€‹

Generalâ€‹

  • Suspend on external power: Sets the screen timeout for when the device is connected to an external power source (i.e. charging). The default value is 5 minutes.

  • Suspend on internal power: Sets the screen timeout for when the device is running on battery power. The default value is 1 minute.

Battery Saverâ€‹

  • Enable: Immediately enables battery saver mode on the device. The default value is false.

    Important

    On Android 8.1, this setting may not persist through a reboot or connection to a charging source.

  • Auto enable: Sets the battery saver mode to be enabled when the battery falls below the specified trigger level. The default value is false.

  • Auto enable trigger level: Sets the trigger level (in battery percentage) for auto enabling the battery saver mode. The default value is 5.

    note

    May be scaled to the closest valid option depending on the Android version.

  • Auto disable: Sets battery saver mode to be automatically disabled when the battery is above 90%. The default value is true.

    Important

    Battery saver auto disable is only available on Android 10+

Wakeup Sourcesâ€‹

  • Wakeup source: The device wakeup source to enable/disable. The default value is None.

  • Enable: Enables the selected device wakeup source. The default value is true.

  • Act directly from suspend: Enables the wakeup source to perform its defined action directly after waking the device from suspend. If disabled, then a second press of the wakeup trigger (after wakeup) is required to perform the action. The default value is false.

Battery Chargingâ€‹

  • Battery charging profile: Represents the charging profile used to manage the charging preferences. This setting is persistent through reboots and factory resets. The default value is Device default preference, which on many devices results in Speed and Capacity preference.

  • Battery charging profile policy: Sets the policy for configuring the battery charging profile. The value selects where the charging profile comes from (device or battery). The default value is Set by device.

  • Enable battery LED: Enables automatic control of the battery LED. The default value is true.

Off-Mode Chargeâ€‹

  • Policy: The policy for automatic exit from charging mode while the device is turned off and being charged. The default value is Auto-Power-On when charge threshold reached.

    note

    Memor 20 devices must be in the cradle and charge threshold must be reached for the off-mode charge Policy to take effect.

  • Threshold The charging threshold (%) for automatic exit from changing mode. Valid values in range of 5-100. The default value is 10.

Charging Sourcesâ€‹

  • Wired cradle enable: Enables charging from wired cradle. The default value is true.

  • Datalogic wireless cradle enable: Enables charging from Datalogic wireless cradle. The default value is true.

  • Other wireless chargers enable: Enables charging from other wireless chargers. The default value is true.

  • USB enable: Enables charging from USB. The default value is true.

    note

    Disabling USB as a charging source will also disable USB client functionality.

Pocket-Modeâ€‹

  • Enable: Enables the Pocket-Mode feature, which can turn off parts of the device when it detects that the display is covered (e.g. in a pocket). The default value is false.

  • Behavior: Selects the behavior when Pocket-Mode is active. The default value is Device lock. The options are:

    • Display off: Turn off the display (the device remains unlocked).
    • Progressive lock: Same as Display Off, but transitions to Device Lock after a timeout.
    • Device lock: Turn off the device (the device is locked).

  • Activation delay: Delay after detection before Pocket-Mode becomes active (500 to 10,000 ms.) The default value is 500 ms..

  • Progressive lock timeout: Timeout before transitioning from Display Off to Device Lock (used with Progressive Lock behavior). The default value is 10 minutes.
