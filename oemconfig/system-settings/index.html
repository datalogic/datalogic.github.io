System Settings

Date and Time​

  • Enable auto time: Enables both time and date to be automatically fetched from the network. The default value is true.

  • Enable auto time zone: Enables the time zone to be automatically fetched from the cellular network. The default value is false.

    note

    Only applies to cellular enabled devices.

  • Select time zone: The timezone to use for the device. The default value is No change.

    Important

    If device is connected to a cellular network and auto time zone is enabled, this setting will be overwritten.

  • Enable 24-hour format: Enables the 24-hour time format for the clock. The default value is false.

NTP​

  • NTP server: The server address to use for NTP time syncing. The default value is an empty string.

  • Secondary NTP server: The secondary NTP server address that will be used if the primary NTP server is not available. The default value is an empty string.

    Important

    A manual reboot will be required on the device before time will be synced.

    tip

    Leave blank for no change to the server address.

  • NTP server connection timeout: Sets the NTP server connection timeout. Valid values are displayed in a single select list. The default value is 10 seconds.

Display​

  • Screen brightness: Sets the screen backlight brightness. A valid value is an integer in the inclusive range of 0 to 100. The default value is 70.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

  • Adaptive brightness: Enables the automatic optimization of brightness level for available light. The default value is false.

  • Auto-rotate: Enables the automatic rotation mode based on device orientation. The default value is false.

  • Font size: The size of the text that appears on screen. The default value is Default.

  • Disable touch screen: Disables input from the touchscreen when set to true. The default value is false.

    Important

    This setting will restrict the user to only using physical buttons when interacting with the device.

Second Display​

Important

Settings only apply to Memor 20 devices.

  • Second display enable: Enables the second display feature on a Memor 20. The default value is false.

  • Add packages: Comma delimited list of package names to enable second display notifications for. The default value is an empty string.

    Example package list

    com.google.android.calendar,com.google.android.gm

  • Remove packages: Comma delimited list of package names to disable second display notifications for. The default value is an empty string.

  • Brightness: Sets the brightness (%) of the second display. A valid value is an integer in the inclusive range of 0 to 100. The default value is 100.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

  • Sleep: Sets the sleep timeout for the second display. The default value is Never.

  • Rotation enable: Enables the text content to be rotated by 180 degrees. The default value is false.

  • Rotation speed: Sets the rotation speed of the second display. The default value is Slow.

  • Enable new app notifications: Enables notifications on second display for all new apps. The default value is true.

Language​

  • Select locale: The locale to apply to the device system. The default value is No change.

Location​

  • Location mode: Used to enable or disable location services (GPS). The default value is GPS on.

Sound​

Volume​

  • Sound source: The sound category to adjust. The default value is None.

  • Volume level: Sets the volume level for the selected sound source. A valid value is an integer in the inclusive range of 0 to 100. The default value is 50.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

Sound/Feedback types​

  • Enable dial pad tones: Enables the dial pad tones when using the phone application. The default value is true.

  • Enable charging sounds: Enables sounds to be played during charging events. The default value is true.

  • Enable touch sounds: Enables sounds effects such as key clicks, lid open, etc. The default value is true.

  • Enable touch vibration (haptic): Enables haptic feedback when interacting with the device. The default value is true.

Touch Mode​

  • Screen sensitivity: Selects the touch screen sensitivity. The default value is Fingers and stylus for the Memor 20 and Fingers mode for the Skorpio X5.

USB​

  • Default USB function: The default USB Transfer Protocol setup every time a USB is connected to the device. Valid values are displayed in a single select list. The default value is None.

  • Current USB function: The USB Transfer Protocol of the current USB transfer. This value may be different from the Default USB function. Valid values are displayed in a single select list. The default value is None.

  • Enable USB debugging: Enables the Android Debug Bridge (ADB) over USB. The default value is false.

  • Reset whitelist: Resets the USB device whitelist. The default value is false.

    note

    Reset will occur before adding new entries.

USB Device Whitelist​

  • Vendor identifier: Vendor identifier for device in the whitelist (0 - 65535). The default value is 0.

  • Product identifier: Product identifier for device in the whitelist (0 - 65535). The default value is 0.

  • Name: Name of the device in the whitelist. The default value is an empty string.

  • Description: Description of the device in the whitelist. The default value is an empty string.

Application Policies​

  • Notify on policy change: Display a notification when an application policy is loaded due to the related application moving to the foreground. The default value is false.

Device Information​

  • Device name format: The format to be used for the device name. The default value is Model.

    Important

    A manual reboot will be required on the device before the setting will take effect.

  • Custom device name base: Custom base part of device name format. The default value is an empty string.

    Important

    A manual reboot will be required on the device before the setting will take effect.
