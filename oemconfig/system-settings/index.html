System Settings

Date and Time​

  • Enable auto time: Enables both time and date to be automatically fetched from the network. The default value is true.

  • Enable auto time zone: Enables the time zone to be automatically fetched from the cellular network. The default value is false.

    note

    Only applies to cellular enabled devices.

  • Select time zone: The timezone to use for the device. The default value is No change.

    Important

    If device is connected to a cellular network and auto time zone is enabled, this setting will be overwritten.

  • Enable 24-hour format: Enables the 24-hour time format for the clock. The default value is false.

NTP​

  • NTP server: The server address to use for NTP time syncing. The default value is an empty string.

    Important

    A reboot is required on the device before time will be synced.

  • Secondary NTP server: The secondary NTP server address that will be used if the primary NTP server is not available. The default value is an empty string.

    Important

    A reboot is required on the device before time will be synced.

    tip

    Leave blank for no change to the server address.

  • NTP server connection timeout: Sets the NTP server connection timeout. Valid values are displayed in a single select list. The default value is 10 seconds.

Display​

  • Screen brightness: Sets the screen backlight brightness. A valid value is an integer in the inclusive range of 0 to 100. The default value is 70.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

  • Adaptive brightness: Enables the automatic optimization of brightness level for available light. The default value is false.

  • Auto-rotate: Enables the automatic rotation mode based on device orientation. The default value is false.

  • 180° rotate enable: Enables 180° rotation. The default value is true.

  • Font size: The size of the text that appears on screen. The default value is Default.

  • Disable touch screen: Disables input from the touchscreen when set to true. The default value is false.

    Important

    This setting will restrict the user to only using physical buttons when interacting with the device.

  • Enable screenshot: Enables screenshot capture. The default value is true.

Desktop Mode​

  • External display mode: Determines the content for the external display. The default value is External screen. The options are:

    • Mirror screen: Copy the device screen. This may produce black bars on the sides to accommodate different orientations.
    • External screen: Use the external screen.
    • Dual screens: Operate each screen separately.
    Important

    A reboot is required on the device before the setting will take effect.

  • Enable taskbar on external display: Enables taskbar on the display when in External Screen mode. The default value is true.

Second Display​

Important

Settings only apply to Memor 20 devices.

  • Second display enable: Enables the second display feature on a Memor 20. The default value is false.

  • Add packages: Comma delimited list of package names to enable second display notifications for. The default value is an empty string.

    Example package list

    com.google.android.calendar,com.google.android.gm

  • Remove packages: Comma delimited list of package names to disable second display notifications for. The default value is an empty string.

  • Brightness: Sets the brightness (%) of the second display. A valid value is an integer in the inclusive range of 0 to 100. The default value is 100.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

  • Sleep: Sets the sleep timeout for the second display. The default value is Never.

  • Rotation enable: Enables the text content to be rotated by 180 degrees. The default value is false.

  • Rotation speed: Sets the rotation speed of the second display. The default value is Slow.

  • Enable new app notifications: Enables notifications on second display for all new apps. The default value is true.

Language and Input​

  • Select locale: The locale to apply to the device system. The default value is No change.

Physical Keyboard​

  • Keyboard layout policy: Determines how the the physical layout is selected. The default value is Default. The options are:

    • Default: Layout is determined by the Android setting.
    • Automatic: Layout is determined by the connected physical keyboard.
    • Custom: Layout is determined by the value of the custom layout setting.

  • Keyboard custom layout: Name of the custom layout to use for the physical keyboard (when the layout policy is set to Custom). This may be set to a layout identifier (e.g. "keyboard_layout_german") or a locale (e.g. "de" or "fr-CA"). The default value is an empty string.

  • Enable keyboard shortcuts: Enables Android keyboard shortcuts. The default value is true.

Virtual Keyboard​

note

GBoard and Voice Typing only apply to GMS devices, and Android Keyboard only applies to AOSP (non-GMS) devices.

  • Enable Gboard: Enables GBoard virtual keyboard. The default value is true.

  • Enable Voice Typing: Enables Voice Typing virtual keyboard. The default value is true.

  • Enable Android Keyboard: Enables Android Keyboard virtual keyboard. The default value is true.

  • Enable custom virtual keyboards: A comma delimited list of custom virtual keyboards to enable. This list is beyond any system standard keyboards. The default value is an empty string.

  • Select virtual keyboard: Component name of the virtual keyboard to use. Set to "default" to use the system default keyboard. The default value is default.

Location​

  • Location mode: Used to enable or disable location services (GPS). The default value is GPS on.

Notifications​

Notification Channels​

  • Notification package: Identifies the package of the notification channel to change. The default value is an empty string.

  • Notification channel: Identifier of the notification channel to change. If empty, all notifications for the package will be impacted. The default value is an empty string.

  • Enable channel: Enables notifications on the specified channel (or all) to be seen. The default value is true.

Notification types​

  • Do Not Disturb policy: Determines the types of notifications that can interrupt the device. The default value is Off. The options are:

    • Off: Do Not Disturb feature is disabled.
    • Allow filtered: Only filtered sources can interrupt.
    • Allow alarms: Only alarms can interrupt.
    • Block all: No sources can interrupt.

  • Enable Home button long press: Enables the response to a long press of the Home button. The default value is true.

  • Enable notification long press: Enables long press expansion of notifications. The default value is true.

  • Enable notification dots: Enables notification dots on app icons. The default value is true.

  • Enable notification LED: Enables the use of the LED for notifications. The default value is true.

Sound​

Volume​

  • Sound source: The sound category to adjust. The default value is None.

  • Volume level: Sets the volume level for the selected sound source. A valid value is an integer in the inclusive range of 0 to 100. The default value is 50.

    note

    Will be automatically scaled to nearest discrete, predefined Android value.

Sound/Feedback types​

  • Enable dial pad tones: Enables the dial pad tones when using the phone application. The default value is true.

  • Enable charging sounds: Enables sounds to be played during charging events. The default value is true.

  • Enable touch sounds: Enables sounds effects such as key clicks, lid open, etc. The default value is true.

  • Enable touch vibration (haptic): Enables haptic feedback when interacting with the device. The default value is true.

Touch Mode​

  • Screen sensitivity: Selects the touch screen sensitivity. The default value is Fingers and stylus for the Memor 20 and Fingers mode for the Skorpio X5.

USB​

  • Default USB function: The default USB Transfer Protocol setup every time a USB is connected to the device. Valid values are displayed in a single select list. The default value is None.

  • Current USB function: The USB Transfer Protocol of the current USB transfer. This value may be different from the Default USB function. Valid values are displayed in a single select list. The default value is None.

  • Enable USB debugging: Enables the Android Debug Bridge (ADB) over USB. The default value is false.

  • Reset whitelist: Resets the USB device whitelist. The default value is false.

    note

    Reset will occur before adding new entries.

USB Device Whitelist​

  • Vendor identifier: Vendor identifier for device in the whitelist (0 - 65535). The default value is 0.

  • Product identifier: Product identifier for device in the whitelist (0 - 65535). The default value is 0.

  • Name: Name of the device in the whitelist. The default value is an empty string.

  • Description: Description of the device in the whitelist. The default value is an empty string.

User Interface​

Block Features​

  • Accessibility: Blocks accessibility settings settings. The default value if false.

  • Airplane mode: Blocks Airplane mode settings. The default value if false.

  • Auto sync application data: Blocks automatic app data sync settings. The default value if false.

  • Do not disturb: Blocks Do Not Disturb settings. The default value if false.

  • Hotspot: Blocks hotspot settings. The default value if false.

  • Android keyboard shortcuts: Blocks Android keyboard shortcut settings. The default value if false.

  • Home button long press: Blocks Home button long press settings. The default value if false.

  • Notification app settings: Blocks app notification settings. The default value if false.

  • Notification dots: Blocks notification dot settings. The default value if false.

  • Reset options: Blocks reset option settings. The default value if false.

Show Features​

  • Airplane mode Quick Settings: Shows Airplane mode in Quick settings. The default value if true.

  • Camera quick access: Enables quick access to the camera (press power button twice). The default value is true.

  • Clipboard data sharing: Controls the level of clipboard data sharing features. The default value is All. The options are:

    • All: All features are available.
    • None: No features are available.
    • Only text manipulation: Only text manipulation (cut, copy, paste, select all) are available.

  • Do not disturb Quick Settings: Shows Do Not Disturb in Quick settings. The default value if true.

  • Emergency SOS: Enables emergency SOS (press power button quickly five or more times). The default value is true.

  • Hotspot Quick Settings: Shows Hotspot in Quick settings. The default value if true.

  • Navigation bar: Shows the Navigation bar. The default value is true.

  • Notification bar: Shows the Notification bar. The default value is true.

  • Notification bar on Lock Screen: Controls level of notifications to display on the Lock Screen. The default value is All. The options are:

    • All: All notifications are displayed
    • None: No notifications are displayed
    • Hide silent: Silent notifications are not displayed

  • Manage notification button: Shows manage button notifications. The default value is true.

  • Managed network notifications: Shows notifications for managed (administratively controlled) networks. The default value is true.

  • Quick Settings Edit: Shows Edit control on Quick settings. The default value if true.

  • Quick Settings Gear: Shows Gear control on Quick settings. The default value if true.

  • Quick Settings Tile List: Comma delimited list of tiles to show in Quick Settings. Set to "default" to use default tiles. The default value is default. Here is a list of all system tiles:

  • Recent apps button: Shows Recent apps button in Navigation bar. The default value is true.

    Important

    A reboot is required on the device before the setting will take effect.

  • Settings search bar: Shows Search bar in Settings app. The default value is true.

  • Split screen apps: Shows split screen app option. The default value is true.

  • Status bar: Shows Status bar. The default value is true.

  • Status bar swipe: Enables Status bar swipe. The default value is true.

  • Status bar swipe on Lock Screen: Enables Status bar swipe on the Lock Screen. The default value is true.

Application Policies​

  • Notify on policy change: Display a notification when an application policy is loaded due to the related application moving to the foreground. The default value is false.

Device Information​

  • Device name format: The format to be used for the device name. The default value is Model.

    Important

    A reboot is required on the device before the setting will take effect.

  • Custom device name base: Custom base part of device name format. The default value is an empty string.

    Important

    A reboot is required on the device before the setting will take effect.

System Settings​

  • Virtual RAM swap size: Size of virtual RAM swap space. Sizes are from zero to 2 GB. The default value is Disabled.
    Important

    A reboot is required on the device before the setting will take effect.
