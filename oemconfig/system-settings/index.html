System Settings

Date and Time#

  • Enable auto time: Enables both time and date to be automatically fetched from the network. The default value is true.

  • Enable auto time zone: Enables the time zone to be automatically fetched from the cellular network. The default value is false.

note

Only applies to cellular enabled devices.

  • Select time zone: The timezone to use for the device. The default value is No change.
important

If device is connected to a cellular network and auto time zone is enabled, this setting will be overwritten.

  • Enable 24-hour format: Enables the 24-hour time format for the clock. The default value is false.

NTP#

  • NTP server: The server address to use for NTP time syncing. The default value is an empty string.
important

A manual reboot will be required on the device before time will be synced.

tip

Leave blank for no change to the server address.

Display#

  • Screen brightness: Sets the screen backlight brightness. A valid value is an integer in the inclusive range of 0 to 100. The default value is 70.
note

Value will be automatically scaled to nearest discrete, predefined Android value. Android 9 and above also contains logarithmic scaling and may not include all values in the lower range.

  • Adaptive brightness: Enables the automatic optimization of brightness level for available light. The default value is false.

  • Auto-rotate: Enables the automatic rotation mode based on device orientation. The default value is false.

  • Font size: The size of the text that appears on screen. The default value is Default.

Location#

  • Location mode: Used to enable or disable location services (GPS). The default value is Gps on.

Language#

  • Select locale: The locale to apply to the device system. The default value is No change.

Sound#

  • Enable dial pad tones: Enables the dial pad tones when using the phone application. The default value is true.

  • Enable charging sounds: Enables sounds to be played during charging events. The default value is true.

  • Enable touch sounds: Enables sounds effects such as key clicks, lid open, etc. The default value is true.

  • Enable touch vibration (haptic): Enables haptic feedback when interacting with the device. The default value is true.

USB#

  • Enable USB debugging: Enables the Android Debug Bridge (ADB) over USB. The default value is false.

Volume#

  • Sound source: The sound category to adjust. The default value is None.

  • Volume level: Sets the volume level for the selected sound source. A valid value is an integer in the inclusive range of 0 to 100. The default value is 50.

note

Will be automatically scaled to nearest discrete, predefined Android value.
