Scan2Deploy Studio 1.3

June 3, 2020 · 2 min read

Scan2Deploy Studio

Scan2Deploy Studio 1.3 is now available for download.

Highlights include new Scanner Settings and Device Settings pages.

Downloads are available in the Releases section on the Scan2Deploy Studio Github page.

Enhancements#

  • Scanner Settings page NEW - configure all the scanner settings and parameters present on your device.

  • Device Settings page NEW - configure Datalogic device settings, including Power and charging, network, and system settings

  • Enterprise Agent, SureFox, and SureLock can now be updated from the Datalogic apps page

  • Added a Copy to Clipboard button on the Save and Print page

  • Apps on Managed apps page are now sorted (alphabetically)

Compatibility#

This release is designed to work with Scan2Deploy Agent v1.18 or later. To update to the latest version of Scan2Deploy agent, check Scan2Deploy on the Datalogic Apps page in Scan2Deploy Studio.

Corrected defects#

  • Removed extra space in URL displayed in the Next Steps section of the Save and Print page

  • Fixed an issue that prevented multiple devices from being able to retrieve a profile from Scan2Deploy Studio HTTP server concurrently

  • Fixed an issue that prevented apps to be added to Studio using the Add Link button on the Install Apps page

  • Added Show Password buttons next to password fields on the Wi-Fi Setup page

Other notes#

  • Backup to Enterprise option removed from Save and Print page.

  • Added debug logging capabilities for development team use.
