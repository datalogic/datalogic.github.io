Android Studio SDK 1.23

September 3, 2020 · 1 min read

Android Studio SDK

Android Studio SDK 1.23 is now available. You can access the documentation here. The online API documentation is available here.

New features:

  • added Cradle APIs to manage key insertion and unlock
  • added APIs to retrieve and store the cradle firmware update policy
  • added 15 WATT as cradle available source power level for Memor 20 devices.
  • added support for ECI Policy
