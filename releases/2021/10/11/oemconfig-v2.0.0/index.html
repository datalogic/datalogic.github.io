OEMConfig v2.0.0

October 11, 2021 · One min read
OEMConfig

Enhancements​

  • Added baroque and viper options to Good read audio mode
  • Added Good read audio channel
  • Added Illumination time
  • Added Scan mode
  • Added Multi Scan Partial results
  • Added Presentation mode
  • Added Barcode settings:
    • Code 39 Aggressiveness
    • Data Matrix Aggressiveness
    • Interleaved 2 of 5 Aggressiveness
    • UPC/EAN Aggressiveness
    • Data Matrix mirror
    • GS1-Data Matrix symbology
    • GS1-QR code symbology
    • ITF-14 symbology
    • OCR symbology
    • Scan2Deploy configuration format
  • Added Off-mode charge
  • Added Memor 20 Dual SIM
  • Added Wi-Fi settings:
    • Power save
    • Roaming profile
    • Channel configuration
  • Added NTP settings:
    • Secondary NTP server
    • NTP server connection timeout
  • Added USB function:
    • Default USB function
    • Current USB function

Corrected Defects​

  • Fixed issue where settings would not apply on Memor 10 Android 8.1
  • Fixed issue where settings would not apply on Memor K
