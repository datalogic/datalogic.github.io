Scan2Deploy Android Schema v1-7-0

September 22, 2020 Â· One min read
Scan2Deploy Android Schema
  • Added new Wi-Fi 6 options to network.mode, including wpa2-eap, owe, wpa3-sae, and wpa3-eap-192.
  • Removed the default for network.mode (was open).
  • Set network.mode to be a required field.
  • Added ftp as a deployment.scheme value.
  • Added ftp-username field, for use when using ftp as a deployment.scheme value.
  • Added ftp-password field, for use when using ftp as a deployment.scheme value.
