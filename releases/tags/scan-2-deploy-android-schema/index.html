Scan2Deploy Android Schema v1-9-0

October 30, 2020 · 1 min read

Scan2Deploy Android Schema

  • Add a new SET_SETTING script command.

  • Add support for loading profiles located in internal storage, SD card and USB flash drives. See storage-type in the deployment section of the Android schema.

Scan2Deploy Android Schema v1-7-0

September 22, 2020 · 1 min read

Scan2Deploy Android Schema

  • Added new Wi-Fi 6 options to network.mode, including wpa2-eap, owe, wpa3-sae, and wpa3-eap-192.
  • Removed the default for network.mode (was open).
  • Set network.mode to be a required field.
  • Added ftp as a deployment.scheme value.
  • Added ftp-username field, for use when using ftp as a deployment.scheme value.
  • Added ftp-password field, for use when using ftp as a deployment.scheme value.

Scan2Deploy Android Schema v1-6-0

September 20, 2020 · 1 min read

Scan2Deploy Android Schema

  • Added default-home section.
  • Added default-home.package-name and default-home.action values.
  • If default-home.action is set to custom-app, then default-home.package-name is a required field.

Note: there was also a lot of white space formatting done in this release, so comparing v1-6-0 to the v1-5-0 version is difficult to parse through.
