Xamarin SDK 1.22.4648

July 28, 2020 · 4 min read
Xamarin SDK

Xamarin SDK 1.22.4648 is now ready to download. You can access the documentation here. The online API documentation is available here. And you can see the package on NuGet here. We did release version 1.21-3280-alpha earlier this year, but as far as official release builds, the last release was version 1.5.88 back in January 2018, so this update is long overdue!

The MAJOR and MINOR version numbers of the Datalogic Xamarin SDK track the version numbers of the Datalogic Android SDK, i.e. Xamarin SDK v1.22.x contains all of the features available in Android SDK v1.22. Therefore, this update includes all the new features introduced in the Datalogic Android SDK from version 1.6 all the way up through version 1.22.

JavaPOS 1.14.059 (1.14.09-059)

February 5, 2020 · 2 min read
JavaPOS API

Enhancements​

  • Firmware update now always sends reset to the device.
  • Added support for DLS Secondary Handheld Serial Number into IHS Parser.
  • Serial port handling refactored and improved.
  • Statistics has been rewritten entirely and now uses a configurable parsing table to parse Information, Health and Statistics responses.
    • Added IHSParser.csv to installation location.
    • Added IHSParserFile configuration option to dls.properties.
  • Added a 9XXX-series custom Direct I/O command to support Cradled Host functionality.
  • Firmware update now immediately exits on a BEL response from the device.
  • Added support for battery specific fields to statistics.

Corrected defects​

  • Corrected an error in WinXXHid.dll nativeSendData causing crashes.
  • Corrected an error in JavaPOSTest causing Cell Phone Scan Exit to call Cell Phone Scan Enter.
  • Added a secondary search for comparing firmware versions.
  • Corrected an error in JavaPOSTest causing the previous weight to remain on screen.
  • Corrected several issues in Statistics implementation.
  • Removed the Edit menu from JavaPOSTest as it was not being used.
  • Removed java.ext.dirs dependencies from the JavaPOS API.
  • Converted Linux USB-OEM implementation to an asynchronous listener.
  • Corrected an encoding error in parsing HDL files.
  • Corrected an error causing Health checks to fail after firmware update.

  • javapos
