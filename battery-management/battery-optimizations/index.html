Battery Optimizations

Enterprise Battery Saver​

Enterprise Battery Saver extends Android's native Battery Saver with enterprise-specific profiles that automatically adjust device behavior based on operational conditions.

Each Profile can reduce battery consumption operating on different device subsystems.

  • CPU/GPU Frequency Limits: Control processor performance to reduce power consumption
  • Screen Brightness: Adaptive or reduced brightness settings
  • Connectivity Management: Selective disable of Wi-Fi, Bluetooth, NFC, cellular, or location services
  • Network Throttling: Set maximum download rate limits
  • Application Control: Blacklist specific applications from running while profile is active

Enabled​

Enables enterprise battery saver feature, which applies a battery saver profile when activation conditions are met.

Ignore power​

Allows Enterprise Battery Saver profiles to remain active even when the device is charging, enabling consistent behavior during specific operational scenarios (e.g., maintaining reduced connectivity during overnight charging).

Profiles​

Manage the battery saver profiles for the device.

To create a new profile:

  1. Tap Profiles > +.

  2. Enter a name for the new profile, then tap OK.

  3. Use the profile settings to configure the profile.

Location​

Select a location to save your profile.

OptionDescription
UserdataStores local app profiles, custom settings, and unmanaged downloads. An Enterprise or Factory Reset wipes this partition entirely to clear local tracking data.
EnterpriseStores secure corporate profiles, Wi-Fi certificates, and MDM logs. This data survives an Enterprise Reset to allow automatic network re-provisioning.
FactoryThe hardware manufacturer's unchangeable configuration profile.
SystemStores global system profiles, Android runtime variables, security rules, and device-wide settings. It is cryptographically signed and mounted as read-only to prevent tampering or app overrides.

Blacklisted apps​

Tap + to add one or more apps to be blocklist.

CPU frequency limit​

Sets the maximum frequency allowed for the CPU.

OptionDescription
NoneNo limitation on the frequency is imposed.
IntermediateThe frequency is limited to an average of the minimum and maximum.
AggressiveThe frequency is forced to the minimum.

GPU frequency limit​

Sets the maximum frequency allowed for the GPU.

OptionDescription
NoneNo limitation on the frequency is imposed.
IntermediateThe frequency is limited to an average of the minimum and maximum.
AggressiveThe frequency is forced to the minimum.

Use power-save governor​

Enables the dynamic CPU speed governor.

Download speed limit​

Sets the maximum download rate (bits/sec) allowed for the device.

Mobile data always active​

Keeps the mobile data on even when connected to a Wi-Fi network.

Adaptive brightness​

Automatically adapts the display brightness according to the ambient light.

Brightness scaling​

Halves the display brightness level when the profile is active.

Disable 5G​

Disables the 5G communication when the profile is active.

Disable Bluetooth​

Disables the Bluetooth communication when the profile is active.

Disable Wi-Fi​

Disables the Wi-Fi communication when the profile is active.

Disable location​

Disables the location service when the profile is active.

Disable mobile network​

Disables the mobile network when the profile is active.

Disable NFC​

Disables the NFC communication when the profile is active.

  1. Tap the save icon to save the profile settings, tap the dustbin icon to discard.

Activation items​

Enterprise Battery Saver can activate specific profiles basing on flexible activation conditions (Activation Items).

To create a new activation item:

  1. Tap Activation items > +.

  2. Enter a name for the new activation item, then tap OK.

  3. Tap Profile name to select the enterprise battery saver profile to apply when the activation condition is met.

  4. Tap Type to select the the type of condition used to activate profile.

OptionDescription
Power SourceApply a profile when connected to specific charge sources (wired cradle, wireless cradle, USB).
Pocket ModeActivate when device detects it's in a pocket or covered.
Battery LevelActivate a profile when charge falls below a specified percentage.
Time IntervalApply different profiles during specified days and time ranges.
ConnectionTrigger on specific connection types.
Always OnDefault profile when no other conditions are met.

  1. Tap the save icon to save the activation item settings, tap the dustbin icon to discard.

Smart Charge​

For all charging profiles that involve reaching maximum battery capacity, Smart Charge optimizes charging patterns based on working hours, reducing the time batteries remain fully charged, one of the main factors contributing to their deterioration.

Enabled​

Activates the smart charge feature.

Discharge delay​

Sets the maximum time (30 minutes to 24 hours) the battery remains at full charge before managed discharge begins.

Work schedule​

Configure start/end times for each day of the week, or set a universal daily schedule.
