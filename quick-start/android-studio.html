Install Android Studio

{% include info-box.html msg='Please notice that the following instructions are valid for Android Studio versions released on the stable, there is no support for the beta channel' %} Download Android Studio from the official website and follow the installation instructions. After installing Android Studio, install the Android SDK through the Android SDK Manager or provide a correct path to one already installed. Since developers are encouraged to target the most recent and updated Android SDK release while developing an application, choose the most new API level released and download the associated platform.

Install Datalogic SDK Add-on

From Android Studio launch window, click on Configure > SDK Manager to open Android SDK Manager

{% include img.html src='install-sdk-as.png' %} {% include img.html src='install-sdk-as-menu.png' %}

Select the tab SDK Update Sites and click the (plus) icon on the right-side toolbar

{% include img.html src='install-sdk-as-sdk-manager.png' %}

In the new window insert the following URL, optionally a name and the press OK

{% highlight none %} {{ site.addonxml_url }} {% endhighlight %}
{% include img.html src='install-sdk-as-site-url.png' %}

Now select the tab SDK Platforms in the Android SDK settings panel and check the checkbox on the bottom right Show Package Details.

{% include img.html src='install-sdk-as-checkbox.png' %}

Under each displayed section, corresponding to the Android releases available, Datalogic SDK v1 Add-on should appear. Choose and install the development kit in accordance with your Android platform of choice. To install it, check Datalogic SDK v1 checkbox and a small icon on the left should appear

{% include info-box.html level='warn' msg='Please notice that you must also install the Android API platform matching your Android version in use for development (i.e. API 23 is required to compile apps supported up to Android 6.0)' %} {% include img.html src='install-sdk-as-before-install.png' %}

Click on the button Apply to install the selected packages and confirm the choice.

{% include img.html src='install-sdk-as-selected-confirm.png' %}

Now, read the corresponding End-User License Agreement (EULA) available in the next window. Accept it, in order to install Datalogic Android SDK.

{% include img.html src='install-sdk-as-selected-confirm-eula.png' %} {% include img.html src='install-sdk-as-selected-confirm-eula-installed.png' %} Once the installation is complete, the Status of the Datalogic SDK v1 Add-on changes from Not installed to Installed. You are now ready to start developing applications compatible with Datalogic devices.

{% include img.html src='install-sdk-as-selected-confirm-installed.png' %}
