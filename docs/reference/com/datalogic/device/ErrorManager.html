Class Overview

This class provides access to error configuration related methods of the terminal. Through this class, Datalogic exceptions can be enabled; otherwise the standard error notification mechanism will be used, relying on simple integer return values. Only the specific exceptions, defined by this SDK, can be controlled can be controlled by this class, standard Java Exceptions are not subject to the configuration specified by the ErrorManager.
